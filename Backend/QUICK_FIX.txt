## 🆘 If Still 404 Error

**Issue**: Backend not listening on port 5000

**Solution**:
1. Is backend running? → Check terminal
2. Port 5000 in use? → Try: `netstat -ano | findstr :5000`
3. Is it stuck? → Kill it: `taskkill /F /IM node.exe`
4. Restart fresh: `npm start`
